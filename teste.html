Instruções para executar o projeto:

  • Clonar este reposítório
  • Ir até o diretório do projeto
  • Criar o arquivo .env a partir do arquivo .env.example
  • Executar o comando: composer install
  • Executar o comando php artisan key:generate
  • Banco de dados:
    • Executar o comando: php artisan migrate --seed (criar as tabelas no banco e executar o seeder para popular o banco de dados)
  • Rodar o servidor: php artisan serve
